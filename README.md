# Java Launcher

Minimal Java Launcher with no known spyware bundled.

Supported platforms:

- Windows
- Linux
- macOS

Schemas in `mlauncher/schemas/` where generated using [jsonschem2pojo](https://www.jsonschema2pojo.org/).
